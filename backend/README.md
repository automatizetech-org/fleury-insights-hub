# Backend (legado / referência)

**Para a VM e um único agendador**, use a pasta **Servidor** na raiz do projeto:

- **Servidor/whatsapp-emissor** — substitui este `whatsapp-emissor`
- **Servidor/server-api** — substitui a pasta `server-api` da raiz
- **Servidor/ecosystem.config.cjs** — sobe os dois com uma tarefa no Agendador

Ver **Servidor/README.md** e **docs/VM_SETUP_COMPLETO.md**.
